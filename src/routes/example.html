Welcome to the world of server-side rendering using Squeeze.

Squeeze works by generating a route from html files in the routes directory. This particular route is named example and was generated from /routes/example.html. The html within each route is compressed before HTMX retrieves it.

You can create your own route by adding a your-route-name.html file in the routes directory. Remember that the HTML head and body elements are already present in the server index, so you can just add the content you wish to render as if it were a component.

div id="content" hx-post="/example" hx-trigger="load" hx-target="#content"

You can alternatively modify this file by editing /routes/example.html

CSS and images are placed in the /static directory.
